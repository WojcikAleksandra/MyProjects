## Structured Data Processing Project - Flight delays analysis

This project focuses on the analysis of flight delays and their underlying causes, including the aircraft production year.

It was developed as part of the *Structured Data Processing (PDU)* academic course as a group project and includes data analysis and visualization implemented in R, with an interactive component built using Shiny.

### Data
**Airline On-Time Performance Data** <br>
Source: https://www.transtats.bts.gov/DatabaseInfo.asp?QO_VQ=EFD&DB_URL=Z1qr_VQ=E&Z1qr_Qr5p=N8vn6v10&f7owrp6_VQF=D

The repository contains only one sample dataset from the above source, covering the year 2008, which is located in the `/data` directory. The other datasets used, which come from different sources, are available in this folder.
